Introduce

Weight Round Robin.

WRR applies one weight ([1, inf]) to every schedule node. And excepts an output schedule chance meets the configued weights.

The basic principle is.

  • Set a weight count (wgt) for every node to record current node selection time. Initialize the count by weight value.
  • Decrease wgt with 1 when scheduler selecting corresponding node.
  • Stop selecting corresponding node when wgt is small than zero.
  • Reload all wgts with corresponding weight when none node can be selecting

DWRR

WRR with deficit.

WRR implements a weighted distribution based as scheduling chance. DWRR extends this weighted distribution to any custom unit. DWRR decreade a weighted unit which user customed instead of 1 when selects a node.

DWRR needs a explict weighted distribution unit. Default is byte.
